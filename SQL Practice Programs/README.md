# 🖥️ SQL Practice Programs
- This folder contains a collection of SQL scripts and programming exercises designed to solve data-driven problems. 

### 📂 Organization
To show a clear progression of logic and technical complexity, the exercises are categorized into the following subfolders:

* **🟢 EASY**
* **🟡 MODERATE**
* **🔴 HARD**

### 🛠️ Key Technical Skills
* **Data Manipulation**: Using `JOIN`, `GROUP BY`, and Subqueries.
* **Advanced Analytics**: Implementing **CTEs** and **Window Functions** (`LAG`, `RANK`, `NTILE`).
* **Data Integrity**: Handling edge cases, NULL values, and date-based logic.
